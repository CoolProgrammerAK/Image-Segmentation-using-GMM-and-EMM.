# GMM-EMM Image Segmentation

Image segmentation using **Gaussian Mixture Models (GMM)** and **Expectation Maximization (EM)** implemented in **Python**. This project supports both **RGB** and **grayscale** images.

## Features

- Segment images into user-defined number of regions
- Works with color (RGB) and black-and-white images
- Simple command-line interface
- Minimal dependencies

## Requirements

- Python 3.x
- NumPy
- scikit-learn
- OpenCV

Install dependencies with:

```bash
pip install numpy scikit-learn opencv-python
